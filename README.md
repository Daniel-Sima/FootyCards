### En localhost:
#### Instructions front/client:
###### Prérequis: Node.js
```bash
cd	footycards/client
```

```bash
npm install
```

```bash
npm start
```
#### Instructions backend/serveur:
###### Prérequis: Golang
```bash
cd  footycards/server
```

```bash
go run ./cmd/api
```
